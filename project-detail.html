> MISSION_OVERVIEW

Đây là đoạn giới thiệu ngắn gọn về game (Elevator Pitch). Ví dụ: Đây là một game nhập vai hành động nơi người chơi phải chiến đấu với binh đoàn bug để cứu lấy production server.

> TECHNICAL_CHALLENGES

Vấn đề: Khi số lượng quái vật lên tới 500 con, FPS bị tụt xuống còn 15fps do Instantiate/Destroy liên tục.
Giải pháp: Tao đã triển khai hệ thống Object Pooling để tái sử dụng các object quái vật và đạn thay vì hủy chúng.

// Object Pooling Implementation Snippet
public GameObject GetPooledObject() {
  for (int i = 0; i < amountToPool; i++) {
    if (!pooledObjects[i].activeInHierarchy) {
      return pooledObjects[i];
    }
  }
  return null;
}
